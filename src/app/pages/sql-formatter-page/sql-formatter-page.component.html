SQL整形ツール

概要

SQLフォーマッターは、読みにくいSQLクエリを整形・可読化するオンラインツールです。JOIN、WHERE、GROUP BYなどのキーワードを適切なインデントで整列させ、コードレビューやドキュメント作成を効率化します。

使い方

  1. テキストエリアにSQLクエリを貼り付けます。
  2. インデントサイズ（スペース数）を選択します。
  3. 整形モード（標準 / tabular右揃え / ミニファイ）を選択します。
  4. キーワードケース・識別子ケース（大文字 / 小文字 / 保持）を選択します。
  5. 「整形」ボタンをクリックすると整形結果が表示されます。

仕様・用語解説

  • 整形モード（mode）
    • standard: 各キーワードを改行・インデントして読みやすく整形します。
    • tabularRight: キーワードを右揃えでタブ状に整列し、SQLキーワードが縦に並びます。
    • minify: 改行・余分なスペースを除去して1行に圧縮します。
  • キーワードケース（keywordCase）
    • upper: SELECT、FROMなどのSQLキーワードをすべて大文字に変換します。
    • lower: SQLキーワードをすべて小文字に変換します。
    • preserve: 入力のケースをそのまま保持します。
  • 識別子ケース（identifierCase）: テーブル名やカラム名の大文字/小文字を変換します。upper / lower / preserve から選択できます。
  • インデントサイズ: 出力するSQLのインデントに使用するスペース数を設定します（例: 2スペース、4スペース）。

ユースケース

  • コードレビュー前にSQLを整理して可読性を向上させたい場合。
  • ORMが自動生成したクエリのデバッグ・解析時。
  • ドキュメント用に整形されたSQLスニペットを作成したい場合。
